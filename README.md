
# SpringBootの練習

SpringBootを練習するために作りました。

# 使っている技術

環境等
- Kotlin
- Gradle
- SpringBoot

Kotlinの静的解析ツール
- ktlint
- detekt

SpotifyAPIを叩くために使っているライブラリ
- Retrofit2
- OkHttp3
- jacksonConverter

今後使う予定の技術
- Redis（APIの返り値のキャッシュ代わりに使いたい）

# 注意

SpotifyAPIを使うためのトークンを環境変数におかないといけない仕様なので今のところ個人使用のためのアプリケーションになっています。
